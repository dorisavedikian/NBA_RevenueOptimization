# 🏀 NBA Revenue Intelligence

An end-to-end sports revenue analytics and business intelligence platform built with Python, SQL, machine learning, and Power BI.

## Project Summary

This project simulates the work of a Lead Data Analyst for a professional basketball organization. It uses a revenue analytics workflow to forecast ticket demand, segment games by demand profile, monitor ticket sales, analyze customer behavior, evaluate funnel performance, and generate executive recommendations.

## Tech Stack

- Python
- SQL / SQLite
- pandas
- scikit-learn
- Power BI
- Git / GitHub

## Analytics Workflow

NBA data and synthetic ticketing data  
→ Python ETL  
→ SQL data warehouse  
→ Feature engineering  
→ K-means segmentation  
→ Regression forecasting  
→ Power BI executive dashboard  
→ Business recommendations

## Machine Learning

### K-means Segmentation

Games are segmented into business demand profiles:

- Premium Demand
- Standard Demand
- Promotion Opportunity
- Inventory Risk

### Regression Forecasting

The model forecasts:

- Tickets sold
- Revenue
- Sellout probability
