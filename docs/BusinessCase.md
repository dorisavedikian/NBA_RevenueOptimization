# NBA Revenue Intelligence — Business Case

## Objective

Build an end-to-end revenue analytics platform that helps a professional basketball organization forecast ticket demand, monitor revenue, optimize pricing, improve customer conversion, and make data-informed business decisions.

## Business Problem

Ticketing teams need to understand which games are likely to sell out, which games need promotional support, how pricing affects demand, and where customers drop off in the purchase funnel.

## Solution

This project creates a simulated sports revenue intelligence platform using Python, SQL, machine learning, and Power BI.

## Key Capabilities

- Ticket revenue reporting
- Inventory and sell-through monitoring
- Web funnel analysis
- Game demand segmentation using K-means
- Ticket demand and revenue forecasting using regression
- Executive recommendations by game

## Business Value

The platform helps leadership answer:

- Which games should receive price increases?
- Which games need promotions?
- Which games are at inventory risk?
- Which games are likely to sell out?
- Where are customers abandoning checkout?
- What revenue should we expect?
